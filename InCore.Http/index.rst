
.. _module_Http:


:index:`InCore Http`
====================

Description
-----------

The Http module provides objects and mechanisms for communicating with HTTP servers. For downloading files the high-level object :ref:`HttpDownload <object_HttpDownload>` can be used. The more low-level :ref:`HttpRequest <object_HttpRequest>` object provides all required methods to perform REST API calls and other kinds of HTTP communications.
